Welcome to Typewriter!

Typewriter is a Web-based writing app that aims to provide a simple and intuitive environment for you to write in. Your work is automatically saved to your browser's LocalStorage, which means that it'll be saved when you reload the page or restart your browser.

To change your preferences, including the theme, font, and more, click the gear icon in the top-left corner.

  • Create a new file by pressing the + button at the top of the screen
  • Download your work as a text, HTML, or Markdown file (ctrl + s)
  • Open a local file in a new tab (ctrl + o)
  • Print your work (ctrl + p)
  • Add headings (ctrl + 1, ctrl + 2, ctrl + 3, ctrl + 4) or remove headings (ctrl + 0)
  • Make text bold (ctrl + b)
  • Make text italic (ctrl + i)
  • Add links (ctrl + k)
  • Strike-through text (ctrl + alt + -)
  • Add bulleted (ctrl + shift + l) or numbered (ctrl + alt + l) lists
