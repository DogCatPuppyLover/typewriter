Welcome to Typewriter!

Typewriter is a simple, Web-based text editor. It autosaves your work to your computer, so you can just write. It doesn't send your work to any other servers.

Some formatting options:

  • Make headings with ctrl + 1, ctrl + 2, ctrl + 3, and ctrl + 4, or remove a heading with ctrl + 0
  • Make text bold with ctrl + b
  • Make text italic with ctrl + i
  • Underline text with ctrl + u
  • Highlight text with ctrl + alt + h
  • Add links with ctrl + k (links only work if you download the document as an HTML file)
  • Strike-through text with ctrl + alt + -
  • Add subscripts and superscripts with ctrl + shift + = and ctrl + shift + -
  • Create an unordered list with ctrl + shift + l
  • Create a ordered list with ctrl + alt + l
  • Insert a horizontal line with ctrl + alt + r
  • Insert an image with ctrl + alt + i
  • Indent text with tab
  • Outdent text with shift + tab
  • Remove formatting with ctrl + \

Other functionality:

  • Download your work as a text or HTML file with ctrl + s
  • Open a file with ctrl + o
  • Print your work with ctrl + p
The preferences page can be accessed by hovering over the top-left corner and clicking the gear icon that shows up. There, you can toggle typewriter sound effects and use custom CSS.
